A feature-rich to-do application with AI integration and mini-games to gamify productivity.

GitHub: DataDefendersWithAI/ToDo-List-Backup

A collaborative web app built with 4 teammates.

  • Full task management: CRUD operations, Eisenhower Matrix prioritization, and calendar integration similar to Google Calendar.
  • AI recommendations: Automatically suggests creating new tasks, continuing existing ones, and optimizing your workflow.
  • Todo Tree mini-game: Your virtual tree grows as you complete tasks, providing a fun gamification layer for productivity.
  • Tech stack: Built with Python Flask, HTML/CSS/jQuery, MySQL, and Tailwind CSS.
  • Deploy: Hosted on DigitalOcean with a Namecheap domain.
