**GitHub**: [DataDefendersWithAI/ToDo-List-Backup](https://github.com/DataDefendersWithAI/ToDo-List-Backup)

Web app "collab" sương sương với 4 đồng âm khác. Một chiếc app nhắc việc full-topping tích hợp AI và mini-game để dụ dỗ user bớt lười.
- Quản lý xịn xò: Có đủ ma trận Eisenhower ưu tiên việc quan trọng, tích hợp đồ thị chạy như Google Calendar.
- AI lo xa: Tự động nhắc nhở, gợi ý phân chia task mượt mà.
- Trồng cây Todo Tree: Mỗi lần làm xong task là cái cây lớn thêm xíu, tăng mood làm việc lên 1000%.
- Đồ nghề: Múa Python Flask, web xài HTML/CSS/jQuery "truyền thống", backend nhét MySQL, bọc thêm Tailwind CSS cho đẹp mã.
- Lên mây: Đã deploy lên DigitalOcean và mua cả domain Namecheap cho oai.
