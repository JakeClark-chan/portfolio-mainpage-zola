# Đồ án trên GitHub

**Website Khảo sát Toxic Positivity**
Website đầu tay tui code lúc 17 nồi bánh chưng. Dùng Python Bottle làm backend, HTML/CSS/jQuery thuần làm frontend, và MySQL làm database. Web này dọn lên PythonAnywhere xài chùa. Mục đích là tạo form xịn xò thu thập feedback về trải nghiệm "toxic positivity" (tích cực độc hại) thay cho Google Forms nhàm chán để làm nghiên cứu khoa học.
*GitHub: (Thâm cung bí sử, hong share được)*

**ToDo List kết hợp Mini-games**
Website thứ hai tui collab với 4 đồng âm khác. Nó là cái app To-Do xịn xò có tag, filter, lịch và mấy cái mini-game xả stress để dụ user làm task chăm hơn. Xài Python Flask, HTML/CSS/jQuery và MySQL. Đợt đó tui chơi sang host hẳn lên DigitalOcean với domain Namecheap đàng hoàng nhe.
*GitHub: [DataDefendersWithAI/ToDo-List-Backup](https://github.com/DataDefendersWithAI/ToDo-List-Backup)*

**Cờ vua dạo (C# .NET)**
Một con game cờ vua C# .NET cho 2 người chơi qua TCP LAN hoặc bắn với bot. Tui xài WinForms vẽ GUI và đá Firebase làm NoSQL database.
*GitHub: [DataDefendersWithAI/ChessWinform](https://github.com/DataDefendersWithAI/ChessWinform)*

**Hàng rào Giám sát Mạng & Web**
Con hàng này làm cho mấy anh admin monitor mạng LAN/WiFi với web services. Cảm hứng từ trò mò báo cháy của Raspberry Pi. Tui mix Python Django (backend) với ReactJS (frontend), rồi gói hết vào Docker cho nó ngầu. Giao diện siêu gọn gàng cho admin soi real-time.
*GitHub: [DataDefendersWithAI/Network-Detector-and-Web-Monitor](https://github.com/DataDefendersWithAI/Network-Detector-and-Web-Monitor)*
