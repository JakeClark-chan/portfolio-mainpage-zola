**GitHub**: [DataDefendersWithAI/Network-Detector-and-Web-Monitor](https://github.com/DataDefendersWithAI/Network-Detector-and-Web-Monitor)

Tool theo dõi "động tĩnh" cho mấy anh em quản trị mạng. Dựa trên cảm hứng từ con Pi.Alert tung hoành giới sysadmin.
- Soi mói từng IP: Thằng nào dám vô ké WiFi hoặc chui vào mạng LAN phát là bắt quả tang và cảnh báo liền.
- Đo nhịp tim website: Tự động chạy nền kiểm tra uptime của các web server xem nó có ngủ quên hay oẳng không.
- Gói gọn tinh tế: Vứt tất cả vào Docker Compose, quăng đi đâu chạy cũng mượt.
- Nguyên liệu chế tác: Lõi Python Django, vung thêm ReactJS ra trình duyệt cho nó ngầu lòi chuẩn UI/UX.
